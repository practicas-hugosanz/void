Política de Privacidad

1. Datos que Recopilamos

VOID almacena localmente en tu dispositivo los datos de sesión, historial de conversaciones y configuración de API. No transmitimos ni almacenamos esta información en servidores externos propios.

2. Uso de los Datos

Los datos almacenados localmente se utilizan exclusivamente para mantener tu sesión activa y recuperar el historial de conversaciones entre sesiones. No vendemos ni compartimos tus datos con terceros.

3. Proveedores Externos

Al usar tu propia API Key, tus mensajes se envían directamente a los servidores del proveedor elegido (Google, OpenAI). VOID no intercepta ni almacena el contenido de dichas comunicaciones. Consulta las políticas de privacidad de cada proveedor.

4. Cookies y Almacenamiento Local

VOID utiliza localStorage del navegador para persistir datos de sesión. No utilizamos cookies de rastreo ni herramientas de analítica de terceros.

5. Tus Derechos

Puedes eliminar todos tus datos en cualquier momento cerrando sesión y limpiando el almacenamiento local de tu navegador. Al cerrar sesión se elimina el token de sesión activo.

6. Seguridad

Implementamos medidas razonables para proteger la información almacenada localmente. Sin embargo, ningún sistema es completamente seguro. Te recomendamos no compartir tu dispositivo con personas no autorizadas.

Preguntas Frecuentes

¿Qué es VOID?

VOID es una interfaz de inteligencia artificial diseñada para ofrecer respuestas precisas y de alto rendimiento. Funciona conectándose a proveedores externos como OpenAI o Google Gemini a través de tu propia API Key.

¿Cómo obtengo acceso?

El acceso está actualmente limitado a través de un programa de whitelist. Puedes solicitar tu plaza desde la sección de acceso anticipado de la página principal. Las solicitudes se revisan de forma manual.

¿Necesito una API Key?

Sí. VOID no almacena ni gestiona claves propias. Para obtener respuestas reales debes configurar tu API Key de OpenAI o Google Gemini desde el panel de ajustes. Sin ella, VOID opera en modo de demostración.

¿Mis datos están seguros?

VOID almacena toda la información localmente en tu dispositivo. No transmitimos ni almacenamos conversaciones en servidores propios. Los mensajes se envían directamente al proveedor que hayas configurado.

¿En qué dispositivos funciona?

VOID es una aplicación web compatible con cualquier navegador moderno en escritorio y móvil. No requiere instalación.
